# termux-zsh
Termux zsh-autosuggestions and zsh-syntax-highlighting in a default termux style bash prompt with red directories for quick shell identification. 

## Installation 
```bash
pkg install git zsh -y && git clone https://github.com/ghostescript/termux-zsh && cd ~/termux-zsh && chmod +x * && ./termux_zsh.sh && cd $HOME
```

## Uninstall 
```bash
cd ~/termux-zsh && ./remove_zsh_config.sh && cd $HOME
```
